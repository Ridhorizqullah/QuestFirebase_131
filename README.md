# MyFirebase - Aplikasi Manajemen Data Siswa

## Deskripsi Proyek
Aplikasi Android berbasis Firebase untuk mengelola data siswa menggunakan Kotlin dan Jetpack Compose.

## Fitur Utama
- ✅ Tambah data siswa baru
- ✅ Lihat daftar siswa
- ✅ Edit data siswa
- ✅ Hapus data siswa
- ✅ Integrasi dengan Firebase Firestore

## Teknologi yang Digunakan
- Kotlin
- Jetpack Compose
- Firebase Firestore
- MVVM Architecture
- Dependency Injection

## Struktur Proyek
```
app/
├── modeldata/       # Model data
├── repositori/      # Repository layer
├── viewmodel/       # ViewModel layer
├── view/            # UI Components
└── ui/              # Theme & UI utilities
```

## Cara Menjalankan
1. Clone repository ini
2. Buka dengan Android Studio
3. Sinkronkan Gradle
4. Jalankan aplikasi
